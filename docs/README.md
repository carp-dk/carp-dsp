## 🎓 Learning Path

### For New Users
1. Read **[../QUICKSTART.md](../QUICKSTART.md)** - Get started in 5 minutes
2. Read **[DSP_DOCUMENTATION.md](DSP_DOCUMENTATION.md)** - Overview section
3. Run the **DEXCOM Retrieval Demo** (simplest)
4. Try the **DEXCOM Analysis Demo** (complete pipeline)

### For Developers
1. Read **Architecture** section in [DSP_DOCUMENTATION.md](DSP_DOCUMENTATION.md)
2. Review **Design Decisions**
3. Explore **Module** details
4. Create your own workflow using the examples

### For Troubleshooting
1. Check **Troubleshooting** section in [DSP_DOCUMENTATION.md](DSP_DOCUMENTATION.md)
2. Check the main project README
3. Open an issue on GitHub if unresolved
